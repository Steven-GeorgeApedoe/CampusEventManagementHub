Campus Event Management System

Welcome to the Campus Event Management System – a comprehensive platform designed to streamline event management and enhance campus life! This project showcases a fully functional system for managing events, including user roles, event creation, and registrations, built with a robust backend and an interactive frontend.

Features
For Students
 Discover Events: Explore a curated list of campus events and activities.

 Register with Ease: Sign up for events with a single click.

 Personalized Dashboard: Keep track of your registered events and event details.

For Event Organizers

 Create Events: Seamlessly create and manage events with key details like date, time, location, and descriptions.

Track Registrations: Monitor attendance and participation statistics.

Admin Controls

 Role-Based Access: Securely manage roles (Admin, Organizer, Student) and permissions.

Comprehensive Event Management: Oversee all events, users, and activities within the system.

Technologies Used

Frontend

React.js: For building a dynamic, user-friendly interface.

CSS: To ensure a visually appealing and responsive design.

Backend

Node.js & Express.js: For creating a robust, scalable backend.

MongoDB: A NoSQL database for efficient data management.

Dotenv: For environment variable management.

Tools

GitHub: Version control and collaboration.

IntelliJ IDEA: Development environment.

Postman: For API testing.

 Key Functionalities

User Authentication

Secure login and registration for students, organizers, and admins.

Passwords securely hashed.

Event Management

CRUD (Create, Read, Update, Delete) operations for events.

Automatic time and date validations.

Role-Based Routing

Access control for different user roles.

Prevent unauthorized access to sensitive actions.

Database Integration

Efficient data storage and retrieval using MongoDB.

Online database hosting for seamless cloud integration.

Frontend-Backend Integration

Real-time communication between the React.js frontend and Express.js backend.
